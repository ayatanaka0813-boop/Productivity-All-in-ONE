Connect Google Calendar

Follow these steps to get a free Client ID from Google. Takes about 5 minutes.

1
Create a Google Cloud project

Go to console.cloud.google.com/projectcreate, give it any name (e.g. "My Workspace"), and click Create.

2
Enable the Google Calendar API

Go to APIs & Services → Library, search for "Google Calendar API", and click Enable.

3
Configure OAuth consent screen

Go to APIs & Services → OAuth consent screen. Choose External, fill in an app name, your email, and click Save and Continue through all steps.

On the Scopes step, add: .../auth/calendar.readonly

On the Test users step, add your own Gmail address.

4
Create an OAuth Client ID

Go to APIs & Services → Credentials, click + Create Credentials → OAuth client ID.

Choose Web application. Under Authorized JavaScript origins, add: null (for local file use) and also http://localhost.

Click Create and copy the Client ID.

5
Paste your Client ID below

It looks like: 123456789-abc...apps.googleusercontent.com
